# P2 Instruction Encodings from Silicon Doc v35 - For Verification

## Purpose
This document captures the exact instruction encodings from Silicon Doc v35 pages 110-118 for cross-verification with other P2 instruction set documentation. Any discrepancies found should be investigated.

## Instruction Format
```
EEEE OOOOOOO CZI DDDDDDDDD SSSSSSSSS
```
- EEEE: Condition code (4 bits)
- OOOOOOO: Opcode (7 bits)
- C: Write C flag
- Z: Write Z flag  
- I: Immediate flag for S
- DDDDDDDDD: Destination (9 bits)
- SSSSSSSSS: Source (9 bits)

## Verification Notes

1. **Encoding Conflicts**: Watch for same opcodes with different CZ bits determining instruction
2. **Bit Field Variations**: Some instructions use NNN fields instead of standard CZI
3. **Sub-opcode Instructions**: Many 1101011 instructions use S field for sub-opcodes
4. **Immediate Flags**: L vs I notation varies - needs standardization check
5. **Missing Instructions**: Cross-check against full instruction reference for completeness
